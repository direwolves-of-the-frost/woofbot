# Woofbot
The official bot for the Direwolves of the Frost Discord server!

# Contributing
Follow the linting rules!

# Deployment
* Run `npm install`
* Fill out the required fields in `.env-example`
* Rename `.env-example` to `.env`
* Run `npm run build` followed by `npm run server`
